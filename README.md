# Hướng dẫn sử dụng: 

  Download file về, giải nén, sau đó cd vào thư mục và chạy lệnh: 
 
  ./zalo với Zalo
 
  ./messenger với Messenger 
 
  App được chạy trên Debian và Kali Linux. App sử dụng tiếng Việt bình thường.
